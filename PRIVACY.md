# Privacy Policy

## AeroSSH - SSH Client for Android

### Data Collection
AeroSSH does NOT collect or transmit any personal data. All SSH credentials are stored locally on your device using secure Android storage.

### Local Storage
- SSH session configurations are stored locally only
- Command history is stored on device only
- Logs are stored in app-private directory

### Network
AeroSSH only connects to SSH servers you explicitly specify. No telemetry, analytics, or tracking is performed.

### Permissions
- **INTERNET**: Required for SSH connections
- **FOREGROUND_SERVICE**: Used to maintain SSH connections in background

### Open Source
All code is open source. You can review the complete source code on GitHub.
